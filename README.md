# Laravel Sanctum API Authentication Skill

An AI agent skill for implementing API authentication with Laravel Sanctum — tokens, SPA auth, and rate limiting.

## Topics Covered

- Token-based authentication (register, login, logout)
- Token abilities/scopes for granular access control
- SPA cookie-based authentication with CSRF
- Rate limiting on auth endpoints
- Token expiration and session management

## Installation

```bash
npx skills add neuraliq/laravel-sanctum-api-auth
# or
php artisan boost:add-skill neuraliq/laravel-sanctum-api-auth
```

## Compatibility

Claude Code, Cursor, Windsurf, GitHub Copilot

## Requirements

Laravel 11+, Laravel Sanctum
